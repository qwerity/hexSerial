# hexSerial


### Gui hex Serial Terminal utility (Qt release)


### Сервисное приложение для тестирования утилит печати на устройствах :

```
* Принтер Spark-115f.
* Устройство "Казначей ФА" с принтером "Custom VKP80II SX".
```


### Программируемые кнопки :

```
* ACK  (0x06)
* NAK  (0x15)
* ENQ  (0x05)
* EOT  (0x04)
* ANY  (0x0a)
```
